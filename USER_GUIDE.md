# 公司訂餐系統使用說明（員工版）

> 這份說明給「中午要點餐的人」看。照著做就會點餐了，不用懂任何技術。

---

## 第一次使用

1. 打開訂餐員給你的**網址**或 **LINE 連結**
2. 看到「公司訂餐 / 請選擇你的名字」
3. 從下拉選單**選你的名字** → 點「進入點餐」
4. ✅ 完成！下次打開會自動記住，不用再選

> ⚠️ **找不到自己名字？** 跟管理員講一聲，他會在後台把你加進去（30 秒搞定）。

---

## 每天怎麼點餐

### 步驟 1：打開連結進入點餐

打開後會看到自己的名字 + 兩個按鈕：

- **🟢 開始點餐** ← 點這個
- 我的訂單

---

### 步驟 2：選你要的品項

進到菜單後會看到：

- **上方分類按鈕**：「🍱 吃的」「🥤 喝的」（如果今天只開一邊就只看到一個）
- **中間品項列表**：每樣東西旁邊有「加入」按鈕
- **底部空白**：購物車一開始是空的

#### 加品項的方法

| 想做的事 | 怎麼點 |
|---------|--------|
| 加 1 份 | 點「加入」按鈕 |
| 多加 1 份 | 加入後，按右邊 [+] |
| 少 1 份 | 按 [−] |
| 完全不要這樣 | 一直按 [−] 到歸零 |

#### 同時點吃的 + 喝的

- 在「🍱 吃的」tab 選便當
- **切到「🥤 喝的」**（看到藍點代表那邊有東西）
- 選飲料
- 兩邊**自動合併**到同一個購物車，**不用分兩次送出**

---

### 步驟 3：檢查購物車

加了東西後，畫面**最下方**會跳出黑色按鈕：

```
🛒 3 項              NT$ 145 ▲
```

點一下會展開，看到分組明細：

```
🍱 吃的  阿婆便當店
  雞腿便當  NT$80 × 1   小計：NT$80
  
🥤 喝的  五十嵐
  紅茶      NT$25 × 1   小計：NT$25
  珍奶      NT$40 × 1   小計：NT$40
─────────────────────────────
共 3 項                  NT$ 145
```

---

### 步驟 4：送出訂單

確認沒問題 → 點購物車旁邊的**「送出」**黑色按鈕

✅ 跳到「我的訂單（今日）」頁，看到剛剛送的訂單

---

## 訂單送出後：5 分鐘內可以改

送出後**有 5 分鐘**可以反悔或改數量。畫面會顯示**藍色倒數**：

```
剩餘可修改
04:32
```

### 想改什麼怎麼辦？

| 想做 | 怎麼做 |
|------|--------|
| 改數量、加減品項 | 點訂單卡上的「**修改 / 取消**」 → 跳出視窗 → 改 → 點「儲存修改」 |
| 整筆取消 | 進「修改 / 取消」→ 找紅色「**取消整筆訂單**」 |
| 不想動 | 等倒數歸零，自動鎖定 |

### 倒數結束後（過了 5 分鐘）

- 修改 / 取消按鈕**會消失**
- 想改？**跟訂餐員講一聲**，他會在後台幫你處理
- 系統會記錄誰改的、什麼時候改、為什麼改

---

## 看我點過什麼

### 「我的訂單」（今日）

從**首頁**或**菜單頁右上角**點「我的訂單」

會看到：

- 黑色卡片：**今日累積金額**
- 下面每筆訂單卡片：吃的、喝的分開顯示
- 還在 5 分鐘內：藍色倒數 + 修改按鈕
- 過了 5 分鐘：唯讀

### 「本週訂單」（週一到週五）

在「我的訂單」頁**右上角**點「本週」連結

會看到：

- **本週你累積點了多少錢**（已扣掉取消的）
- 一週的所有訂單列表
- 每筆訂單可以點進去看詳細

> 💡 訂餐員每週五用這個金額跟你收費。

## 常見問題

### Q：找不到我的名字
跟管理員講，他在後台加你進系統。約 30 秒。

### Q：菜單顯示「今日尚未開單」
訂餐員還沒選今天的店家。等一下再回來看。

### Q：菜單顯示「已結單」
- 訂餐員已經把訂單彙整給餐廳了，**今天不能再加了**
- 想加？只能拜託訂餐員看餐廳能不能再多收一單

### Q：菜單顯示「已取消」
今天的訂餐被取消了（通常是人數太少不划算外送）。會顯示原因。

### Q：訂單金額跟我預期的不一樣？
- 餐廳實際收的錢可能跟菜單不同（**缺貨換別的、餐廳漲價沒更新菜單**）
- 訂餐員會在後台幫你**修正成實收金額**
- 重新整理頁面就會看到變動 + 修改原因
- **週結算用修改後的金額**，不會多收

### Q：我點了系統會自動扣款嗎？
**不會**。
- 點完不會立刻扣錢
- **訂餐員每週五**用本週累積金額跟大家收（現金 / 轉帳，依公司規定）

### Q：我可以改同事的訂單嗎？
**不行**。一個身份只能改自己的。同事忘了點？請他自己點，或請訂餐員幫忙。

### Q：怎麼切換成同事的身份？
**只在公用裝置上才需要這樣**。
- 首頁最下方「**不是我？切換身份**」小字
- 點下去重選名字

### Q：我可以一天點兩次嗎？
**可以**。
- 送出第一筆後，**回菜單繼續點**就會產生第二筆
- 兩筆是獨立的，5 分鐘可改的時鐘**各自跑**

### Q：我下午突然想加飲料，但中午已經結單了
- 喝的場次有可能還沒結（看訂餐員怎麼設）
- 如果**「🥤 喝的」還顯示「進行中」**就還能點
- 如果已結單就只能等明天

### Q：手機上字太小 / 按鈕太小
這個系統設計成手機優先，理論上不會。如果還是有問題，反映給管理員處理。

## 找誰幫忙？

| 問題類型 | 找誰 |
|---------|------|
| 找不到名字 / 帳號相關 | 管理員 |
| 訂單金額對不起來 / 過 5 分鐘想改 | 訂餐員 |
| 餐廳缺貨想換餐 | 訂餐員（在電話裡跟他講就好，他會幫你改）|
| 系統打不開 / 跳出錯誤訊息 | 訂餐員或管理員（**截圖**給他們最快） |
| 「我這週收費怎麼算的？」 | 訂餐員 |
